# yefan-cli

## 📖 简介

- 用于快速搭建前端项目的命令行工具

## 📕 模板

- ✅ [一键生成基于 Vue/React 等框架的开发模板](https://www.npmjs.com/package/yefan-cli)

## 📦 安装

```bash
npm install yefan-cli -g
```

# 🚩 使用

```bash
# 使用模板创建新项目

yefan create

# 执行项目文件夹名称并选择模板创建新项目

yefan create [projectName]

# 更新脚手架到最新版本

yefan update

# 查看脚手架版本

yefan -v
yefan --version
```
